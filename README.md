# 💰 Expense Tracker App

## 📌 Overview
This is a simple Expense Tracker web application built using Streamlit.  
It helps users record and manage their daily expenses easily.

---

## 🎯 Problem Statement
Many students do not track their daily expenses properly, leading to overspending.

---

## 💡 Solution
This app allows users to:
- Add expenses
- Categorize spending
- View total expenses

---

## 🚀 Features
- Add expense amount, category, and date
- View all expenses
- Total spending calculation

---

## 👥 Target Users
- Students
- Young professionals

---
